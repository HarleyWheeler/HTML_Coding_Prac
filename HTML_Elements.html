Elements

An element is everything between the the start and end tag.

Nested HTML Elements

Elements can contain other elements, such as p being contained within h1, within body, within html

Never skip the end tag

Even though some HTML elements will display without end tags, it is bad practice to not include end tags.

Empty HTML elements

Tags such as breaks
can be inserted mid-line

to separate items.

Tags are not case sensitive, so something like either 'p' or 'P' could be used.
